# Fra2mo Description Package

This package contains urdf descriprion for the Fratomo robot used for the Robotics Lab course

### Launch files

- **Simulazione Gazebo:**
  ```bash
  ros2 launch fra2mo_description fra2mo_gazebo.launch.py
  ```

- **Visualizzazione RViz:**
  ```bash
  ros2 launch fra2mo_description fra2mo_rviz.launch.py
  ```
